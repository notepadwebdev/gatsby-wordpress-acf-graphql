# ContentBlock

A block of content that includes an image, title and WYSIWYG content area.   
The image will be aligned left or right on larger devices.

<ArgsTable of={ContentBlock} />

## Stories
---

### Default / Image Left
The (optional) alignment prop is set to `image-left` by default and the image will be aligned left on larger devices if this prop is left unset.
<Canvas>
  <Story name="Default / Image Left" args={Default.args}>
    {Template.bind({})}
  </Story>
</Canvas>

### Image Right
If the alignment prop is set to `image-right` then the image will be aligned right on larger devices.
<Canvas>
  <Story name="Image Right" args={{
    ...Default.args,
    alignment: `image-right`
  }}>
    {Template.bind({})}
  </Story>
</Canvas>
